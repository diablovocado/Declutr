# Declutr RC1 Enterprise Administrator Guide

Guide for cluster administrators managing Declutr in enterprise environments.

## Administrative Capabilities

- **Web Admin Portal**: Accessible at `/admin` for viewing live telemetry, worker pools, cache stats, and distributed traces.
- **Enterprise Portal**: Accessible at `/organization` for tenant switching, RBAC role editing, team/department group creation, workspace classifications, and governance policy configuration.
- **Developer Portal**: Accessible at `/developer` for API key management, OAuth 2.1 client apps, webhook DLQ monitoring, and interactive OpenAPI testing.
- **Extension Marketplace**: Accessible at `/marketplace` for extension management, digital signature verification, and publisher releases.
