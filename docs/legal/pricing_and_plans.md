# Declutr Subscription Tiers & Pricing Model

Declutr provides flexible deployment options ranging from open self-hosted to enterprise multi-tenant cloud deployments.

## Subscription Tiers

| Tier | Price | Highlights |
|---|---|---|
| **Community (Self-Hosted)** | Free (Open Source) | Unlimited Vaults, Hybrid Search, Offline Sync, CLI, Local AI |
| **Pro Personal** | $12 / mo | Managed Cloud Storage, Grounded RAG Copilot, Webhooks |
| **Enterprise Cloud** | $29 / user / mo | Multi-Tenancy Organizations, SSO/SAML, Custom Policies, Audit Logs, Dedicated Support |
