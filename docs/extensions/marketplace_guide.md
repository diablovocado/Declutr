# Declutr Extension Marketplace Guide

The Extension Marketplace provides a curated storefront for discovering, installing, rating, and managing extensions.

## Marketplace Features

- **Browsing & Search**: Search by name, author, capability, or filter by category (`AI`, `Productivity`, `Automation`, `Storage`, etc.).
- **Verified Publisher Badges**: Digital signature verification and publisher identity validation.
- **Ratings & Reviews**: User ratings (1-5 stars) and review comments.
- **Lifecycle Management**: Enable, Disable, Rollback to previous version, Uninstall, or Repair.
