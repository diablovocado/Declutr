# Declutr Extension Publishing Guide

Developers can publish extensions to the Marketplace via the Publisher Developer Portal (`/marketplace/publisher`).

## Publisher Workflow

1. Register Publisher Profile (Website, Contact Email).
2. Build extension bundle using `@declutr/extension-sdk`.
3. Sign manifest with Publisher RSA/ECDSA Private Key.
4. Upload Release Version bundle via Publisher Portal.
5. Track download analytics, active user counts, and crash reports.
