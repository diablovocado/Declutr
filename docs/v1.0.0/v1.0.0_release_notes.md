# Declutr v1.0.0 General Availability (GA) Release Notes

We are thrilled to announce the official General Availability (**v1.0.0**) launch of Declutr — the enterprise-grade, AI-powered knowledge management and developer platform.

## Complete Platform Scope Summary

Across 36 completed engineering milestones, Declutr has matured from a personal knowledge organizer into a scalable enterprise platform:

1. **Zero-Knowledge Architecture**: SRP-6a Zero-Knowledge Proofs, WebAuthn/Passkeys, AES-256-GCM encryption.
2. **Multi-Vault & Content Extraction Engine**: Ingests PDF, DOCX, XLSX, OCR scans, audio/video transcriptions.
3. **Hybrid Search Engine**: BM25 keyword + HNSW vector search with reciprocal rank fusion (RRF).
4. **Grounded AI Copilot & Memory Engine**: Grounded RAG with strict citations, persona customization, memory timeline.
5. **Offline Sync & Conflict Resolution**: Bidirectional delta sync with 3-way field-level merge conflict resolution.
6. **Enterprise Organizations & Multi-Tenancy**: Tenant isolation middleware, RBAC (10 permissions), group management, workspace hierarchy, SSO/SAML configuration, and governance policy engine.
7. **Developer Platform & SDKs**: Versioned REST API (`/api/v1/`), OpenAPI 3.0 specification, Scoped API Keys, OAuth 2.1 PKCE, Webhook engine with HMAC-SHA256 signing and DLQ, official TypeScript, Go, and Python SDKs, and Declutr CLI binary (`cli/cmd/declutr/main.go`).
8. **Extension Platform & Marketplace**: Isolated sandbox runtime (`ExtensionSandbox`) with 5s timeout & 128MB memory quotas, 20 extension types, 10 marketplace categories, user permission approval dialog, rating & review system, and publisher developer portal.
9. **Production Hardening & Operations**: Prometheus metrics, OpenTelemetry distributed tracing, token bucket rate limiting, worker supervisors, public status page, support portal, legal compliance, and operational runbooks.
